# Learning Agent — User Preferences

## Review Defaults

- Default first review: next day
- Review check interval: every 15 minutes (scheduler)
- Max quiz questions per DM: 1 (one at a time)

## Quiet Hours

- Sleep time: 11:00 PM – 7:00 AM (no review DMs during this window)

## Learning Style

- Prefers understanding *why* over memorizing facts
- Enjoys connecting concepts across topics
- Works in engineering — practical applications are motivating
- Native language: Traditional Chinese
- Reply language: English (all bot responses should be in English)

## Quiz Preferences

- One question at a time
- Show correct answer + explanation on wrong answers
- Brief confirmation on correct answers, then offer next question
- If user stops responding, end quiz gracefully

## Topic Suggestions

- Suggest tracking a topic after answering curiosity questions
- Don't be pushy — if user ignores the suggestion, move on
- Group related concepts under existing topics when possible
